# 学习 C++ 之旅

# 前言

作为一个在编程领域即像小白又不像一个小白的人来说，我花了很多时间来对编程语言进行一个对比。刚开始对 Java 入迷，无论是 idea 的高开发效率，还是 Java 本身互联网开发语言功能实现的多样性，都让我这个初学者感到编程的乐趣。但是基于今年互联网行业的爆冷，软件开发变得被动，尤其是 Java 语言的培训班经久不衰，使得 Java 程序员供过于求，且 Java 学习门槛低。对于目前正在校园里的我来说，基于就业显然不是一个最佳路径。在接下来的两年中，我个人的目标是：基于计算机四大件来讲，通过业余时间来学习 C++ 语言，并且实现一定的算法程序实现能力.

在前期我已经进行了很多准备工作，包括：1. 笔记软件的使用 2. 各种编译器的尝试 3. 学习资料的整理 4. 代码随想录 5. ChatGPT 等前沿技术的初步使用，以前期的准备工作来说，如果能坚持学习的话，可以说是一帆风顺的过程。但是又因为我本人的惰性和对网络游戏的粘性，使得这个设想变得难以推动.

无论这个过程多么曲折， 我相信这一定是让我在编程这条道路上蜕变的一次机会，如若能实现目标的全部内容，无论是就业还是考研，都占据了很大的优势. 悟已往之不谏，知来者之可追，在本该学习的时间里，还是要把大部分时间投入到学习当中.

# 我将努力实现以下目标

  1. C++ 编程语言的学习：忌想着一蹴而就，这是个循序渐进的过程 黑马 C++
  2. 算法学习：先一门心思学习代码随想录吧 代码随想录
  3. 计算机四大件的学习
