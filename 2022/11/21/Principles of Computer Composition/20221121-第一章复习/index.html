# 第一章复习

  1. 机器字长：也就是我们说的 64 位，32 位计算机，机器一次能处理的二进制位数，由运算器和寄存器的位数决定

  2. 时间指标:

    1. 主频 f: 时钟频率，与 CPU 的运算能力不是唯一挂钩的

    2. 时钟周期 T=1/f

    3. CPI: 执行一条指令平均需要的时钟周期数

      ​ CPI=m/ic m: 一段程序的时钟周期总数，ic: 指令总数

    4. CPU 时间：执行一段程序需要的时间

      T (cpu 时间)=CPI*IC *T

    5. MIPS: 每秒执行的指令条数

      计算：MIPS=f/CPI*10^6

    6. MFLOPS: 计算机每秒执行的浮点数运算

本章重点：CPU 时间的计算
