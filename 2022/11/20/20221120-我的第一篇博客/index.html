我的第一篇博客

# 前言

谨以此文纪念我的博客网站的建立

# 正文

一直以来，我都对计算机方面充满兴趣。也被电子笔记系统化学习所吸引，一直使用 obsidian 进行笔记整理的我，在一次偶然的机会中认识到了个人博客这个东西，就让这个网站记录我从大二开始的生活吧，温馨提示，生活包括了一下几项：1. 计算机专业知识的整理 2. 生活中的碎碎念 3. 本人的恋爱日常 (出于博客专业性的设定，这一项有待斟酌)

在开始搭建网站的这三天里，基本上白天我都在上课，也只有利用自己课间的闲暇时光，去处理一些技术上的问题。可曾知，命令行中的报错总是在你一遍又一遍地输入指令后，仍然死皮烂脸，接踵而至；网页配置文件的修改更是繁琐沉重，一不留心少个空格，或者拼写错误就足够折磨；Github 在国内的访问连接也是时好时断，网络超时中断造成上传失败，一遍又一遍在挫败我的耐心与信心……

然而，雄关漫道真如铁，而今迈步从头越！终于在今天晚上，这一切的一切，总算是告一段落。尽管博客网站上还有很多功能，如” 标签 “” 分类 “板块的补充，” 留言板 “的搭建，以及访问主题界面的进一步优化等等，还未付诸实现。但是有什么要紧的呢，当我望着这个充满缺陷但是至少” 像模像样 “的网站时，心中仍旧是充满了无比欣慰，甚至激动无比。就像是儿童时代，小孩子在完成了爸妈交给的事情后，那种欣喜若狂，认为自己是长大了，以后在家里也可以” 顶天立地 “一样……

当然，我深知，在这个博客项目的很多地方，自己无非是借用了网上大佬们的模板。在网页前端设计方面上一窍不通的我，这个博客的搭建也不过是停留在配置文件上的删改，这与一个真正意义上的博客网站创建、编写、设计、调试、上传还相距甚远，所以无论是知识的学习，还是实际的操作，我要努力的地方还有很多。可以说，来日方长，未来可期，要走的路还很漫长呢！

从长远上说，这个博客网站的搭建，其实是为我后续在编程方向的学习做铺垫的。慢慢地，我会进一步去完善她的内容，丰富上面的功能，修改一些错误。与此同时，我也会陆陆续续地在上面，上传自己在计算机方面的学习历程，比如说分享一些有趣的 C 语言题目，还有自己目前正在学的数据结构课程。当然，未来肯定也不会止于这些。本质上，我也想依靠这个博客的建立与完善，进一步督促自己的学习和坚定朝计算机方面发展的动力，并去完善自己，充实自己，也成就自己！

倘若，在这条建设网页，琢磨” 小玩意儿 “，折磨自己与 bug 对抗的路上…… 能够学到更多知识，收获一些工作感悟，有所建树，甚至能够为这个社会、国家带来一些小贡献时，这又何尝不是人生一件乐事呢！

# 后记

很荣幸你能浏览这个网页，并看完这篇文章。如果你有什么想法或者建议，可以通过网站上的联系方式与我交流。如果你也对此抱有兴趣，本次编写博客网页的所有源代码，也均在我的 Github 账号主页的项目中能够找到，欢迎大家相互交流学习！

最后，创作不易，如果这篇博文给你带了帮助，也希望你能够把网站地址分享给更多的朋友，让本平台能够被更多的人所看到，而读者的支持也正是我们创作者为之奋斗的动力！当然，也欢迎大家点击下方标有” 赏 “字的按钮，进行打赏和支持，这亦是对我们创作的最大肯定！

再次感谢您的浏览和支持
